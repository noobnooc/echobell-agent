# Prompt Kit

## Minimal completion prompt

```text
Use $echobell-notify and send me an Echobell notification when this task completes.
```

## Completion with urgency

```text
Use $echobell-notify and send a time-sensitive Echobell notification when the migration plan is ready.
```

## Failure-only notification

```text
Use $echobell-notify and only notify me if the release validation fails.
```

## Completion and failure split

```text
Use $echobell-notify. Send active on success and calling on failure.
```

## Explicit payload style

```text
Use $echobell-notify. When you notify me, keep the title under 60 characters, keep the body to two lines, and include a link only if there is a useful artifact to open.
```
